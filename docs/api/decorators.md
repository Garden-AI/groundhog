# Decorators

Groundhog provides three decorators:

- `@hog.function()` - For remote-executable functions
- `@hog.method()` - For remote-executable class methods
- `@hog.harness()` - For local orchestrator functions

## @hog.function()

::: groundhog_hpc.decorators.function
    options:
      show_root_heading: true
      heading_level: 3

## @hog.method()
